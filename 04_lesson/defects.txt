=== Дефекты в StringUtils ===

[Дефект #1] capitalize() не обрабатывает строки с ведущими пробелами
Серьезность: Medium
Описание:
- Метод не делает первую букву заглавной, если строка начинается с пробелов
Шаги воспроизведения:
1. Вызвать capitalize(" skypro")
Ожидаемый результат: " Skypro"
Фактический результат: " skypro"

[Дефект #2] trim() не удаляет whitespace символы кроме пробела
Серьезность: Low
Описание:
- Метод не удаляет табуляции и переносы строк в начале строки
Шаги воспроизведения:
1. Вызвать trim("\thello")
Ожидаемый результат: "hello"
Фактический результат: "\thello"

[Дефект #3] trim() вызывает исключение при None
Серьезность: High
Описание:
- Метод не обрабатывает None входные данные
Шаги воспроизведения:
1. Вызвать trim(None)
Ожидаемый результат: ""
Фактический результат: AttributeError

[Дефект #4] contains() возвращает True для пустого символа
Серьезность: Medium
Описание:
- Метод некорректно обрабатывает пустую подстроку
Шаги воспроизведения:
1. Вызвать contains("abc", "")
Ожидаемый результат: False
Фактический результат: True

[Дефект #5] contains() не обрабатывает None
Серьезность: High
Описание:
- Метод вызывает исключение при None входных данных
Шаги воспроизведения:
1. Вызвать contains(None, "a")
Ожидаемый результат: False
Фактический результат: AttributeError

[Дефект #6] delete_symbol() не обрабатывает None
Серьезность: High
Описание:
- Метод вызывает исключение при None входных данных
Шаги воспроизведения:
1. Вызвать delete_symbol(None, "a")
Ожидаемый результат: ""
Фактический результат: AttributeError
